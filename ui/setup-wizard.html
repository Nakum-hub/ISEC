👋 Welcome to ISEC
ISEC is an enterprise forensic evidence collection platform. This setup wizard will activate your license, verify your system, and configure the application in under 2 minutes.

What ISEC does:
🔍 Collects system logs, browser history, network connections and file metadata
🔐 Signs every record with HMAC + SHA-256 hash chain (court-admissible)
📋 Generates forensic PDF reports with chain of custody documentation
⚖️ Maps evidence to NIST CSF 2.0 compliance framework
